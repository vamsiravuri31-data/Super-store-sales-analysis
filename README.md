# Sales and Profit Analysis of Superstore Dataset

# Tools Used
- Python (Pandas)
- Power BI

# Project Overview
Performed end-to-end data analysis on a retail dataset including data cleaning, EDA, and dashboard visualization.

# Key Insights
- Technology category has highest sales
- High discounts reduce profit
- Some products generate losses
- Regional and seasonal trends identified

# Dashboard
Power BI dashboard provides interactive insights using charts and filters.
